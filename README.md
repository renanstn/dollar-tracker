# dollar-tracker

## Descrição

Meu app de tracking do preço do dólar diário.

Consome a API https://awesomeapi.com.br para obter os valores.

## Setup

```py
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
# Optional for development only
pip install black
pip install isort
```
